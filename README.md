Repositorio personal de formación y referencia para el trabajo en **Araiz Suministros Eléctricos**.

## ESTRUCTURA DEL PROYECTO

```
ARAIZ/
├── 01_CATALOGO/
│   └── familias_productos.md       → Todas las familias de productos de Araiz estructuradas
│
├── 02_BASE_CONOCIMIENTO/
│   ├── automatizacion_industrial.md → PLCs, variadores, contactores, sensores... explicados
│   ├── material_electrico_basico.md → Protecciones, cable, canalizaciones, armarios...
│   └── marcas_principales.md        → Schneider, Siemens, ABB, SMC, Legrand, Hager...
│
└── 03_TELEMATEL/

```

---

## CÓMO USAR ESTE PROYECTO

1. **Empezar por:** `01_CATALOGO/familias_productos.md` — para tener el mapa general de lo que vende Araiz.
2. **Profundizar en:** `02_BASE_CONOCIMIENTO/` — para entender qué es cada cosa.
3. **Prepararse para:** `03_TELEMATEL/` — para llegar con ventaja el día que den acceso al ERP.

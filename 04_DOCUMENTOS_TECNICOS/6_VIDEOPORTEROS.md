# VIDEOPORTEROS Y SISTEMAS DE ACCESO
### Guía técnica para Aurora

---

## COMPONENTES BÁSICOS DE UN VIDEOPORTERO

| Componente | Función | Dónde está |
|---|---|---|
| **Placa de calle** | Cámara + pulsadores + micrófono | Exterior, en la puerta |
| **Monitor / Telefonillo** | Recibe imagen y voz, permite apertura | Interior de la vivienda |
| **Alimentador DIN** | Fuente de alimentación del sistema | En el cuadro eléctrico |
| **Distribuidor de planta** | Reparte la señal a varias viviendas | Registros de cada planta (edificios) |
| **Abrepuertas eléctrico** | Relé que abre la cerradura a distancia | En el marco de la puerta |
| **Bus 2 hilos** | Cable que conecta todos los elementos | Dentro de las paredes |

---

## TECNOLOGÍA ACTUAL — SISTEMAS 2 HILOS NO POLARIZADOS

El estándar profesional actual son los **sistemas digitales de 2 hilos no polarizados** (no importa cuál de los dos hilos va a qué terminal). Esto simplifica enormemente el montaje y elimina errores de conexión.

**Marcas líderes en este sistema:**
- **Fermax DUOX** (español, muy presente en España)
- **Tegui 2 Hilos** (también muy usado)
- **Comelit BUS**
- **Aiphone**
- **Golmar**

> **Por qué importa para Araiz:** los sistemas 2 hilos tienen alta rotación de accesorios y recambios. Los distribuidores de planta, cables BUS, alimentadores y monitores son pedidos recurrentes.

## INSTALACIÓN EN VIVIENDA UNIFAMILIAR (Kit)

El **Kit completo** es el producto de mayor rotación para reformas rápidas. Incluye todo lo necesario en una caja.

### Fases de instalación:

1. **Replanteo:** la placa de calle se coloca entre 1,40m y 1,60m del suelo
2. **Cableado:** cable BUS de 2 hilos (par trenzado para tiradas largas)
3. **Montaje físico:** caja de empotrar en exterior + soporte monitor en interior
4. **Conexión:** 2 hilos entre placa, monitor y alimentador
5. **Pruebas:** verificar audio, imagen y apertura de puerta

### Materiales típicos de un kit unifamiliar:

| Artículo | Cantidad | Observación |
|---|---|---|
| Placa de calle | 1 ud | Con cámara integrada |
| Monitor interior | 1 ud | Pantalla 4,3" a 7" |
| Alimentador DIN | 1 ud | Fuente 18V DC para el BUS |
| Abrepuertas eléctrico | 1 ud | 12V DC / AC |
| Caja de empotrar placa | 1 ud | Para empotrar en pared exterior |
| Cable BUS 2x0,5 o 2x0,75mm² | X metros | Par trenzado |

## INSTALACIÓN EN BLOQUE DE VIVIENDAS (Sistema Colectivo)

Mayor complejidad: hay que gestionar la distribución vertical y el direccionamiento de cada vivienda.

### Fases de instalación:

1. **Troncal (Montante):** cable BUS principal que recorre el edificio verticalmente desde el portal hasta cada planta. Suele ir desde el RITS (Recinto de Telecomunicaciones Superior).

2. **Distribución de planta:** en cada piso se instala un **Distribuidor de Planta**:
   - Protege la señal si hay cortocircuito en una vivienda
   - Permite derivaciones hacia cada monitor

3. **Conexión de viviendas:** desde el distribuidor de planta hasta el RTR (Registro de Terminación de Red) dentro del piso.

4. **Programación y direccionamiento (paso crítico):**
   - Se asigna un **código único** a cada monitor (número de piso/puerta)
   - La placa de calle sabe qué monitor llamar al pulsar cada botón
   - Se programa desde el monitor o mediante pulsaciones específicas en la placa

5. **Balanceo de impedancia:** en edificios de muchos pisos hay que configurar cargas de terminación en los distribuidores para evitar pérdida de calidad en las plantas más lejanas.

### Materiales típicos de un sistema colectivo (10 viviendas):

| Artículo | Cantidad |
|---|---|
| Placa de calle modular (con N botones) | 1 ud |
| Monitores interiores | 10 uds |
| Alimentador DIN | 1-2 uds (según longitud) |
| Distribuidores de planta | 1 por cada 2-4 pisos |
| Cable BUS 2 hilos (par trenzado) | Según metros de montante + distribución |
| Cajas de empotrar | Según proyecto |

## TIPOS DE PLACA DE CALLE

| Tipo | Cuándo se usa |
|---|---|
| **Monoblock** | Viviendas unifamiliares y edificios pequeños. Todo integrado en una pieza. |
| **Modular** | Edificios con muchas viviendas. Se combinan módulos (cámara + N botones + teclado). |
| **Con teclado numérico** | Edificios grandes donde los visitantes marcan el número de piso. |
| **Con lector de tarjeta/proximidad** | Control de acceso avanzado (oficinas, garajes). |

## ACCESORIOS IMPORTANTES

| Accesorio | Para qué sirve |
|---|---|
| **Visera de protección** | Protege la placa de lluvia y reflejos solares en exteriores expuestos |
| **Marco de superficie** | Para instalar la placa sin empotrar |
| **Fuente de alimentación adicional** | Para tiradas largas de cable o muchos monitores |
| **Módulo de apertura adicional** | Para segunda puerta o garaje |
| **Adaptador de portero analógico** | Para conservar el portero viejo y añadir función de video |

## PARA GESTIÓN — CLAVES DE UN PEDIDO DE VIDEOPORTERO

**Preguntas que debes hacer al cliente:**

1. ¿Unifamiliar o colectivo (cuántas viviendas)?
2. ¿Instalación nueva o sustitución de uno existente (qué marca/sistema hay)?
3. ¿Cuántas plantas y cuántas viviendas por planta?
4. ¿Se necesita control de acceso (tarjeta, código)?
5. ¿Sistema convencional o IP (con app de móvil)?
6. ¿Hay ya canalización/tubo pasado o hay que preverlo?

> **Tip de gestión:** los kits de vivienda unifamiliar son los de mayor rotación (reformas, sustituciones). Los sistemas colectivos son menos frecuentes pero de mayor importe por pedido.

# Instrucciones del proyecto Araiz

## Comando personalizado: "Keep in touch"

Cuando el usuario escriba únicamente la frase **"Keep in touch"**, debes ejecutar automáticamente este protocolo de cierre paso a paso:

1. **Auditoría y Limpieza:** Revisa los archivos tocados en la sesión. Elimina código comentado inútil y archivos temporales de prueba.
2. **Guardado Final:** Haz un commit en Git con el mensaje "Cierre de sesión: [Resumen de lo logrado hoy]".
3. **Relevo:** Actualiza el archivo `SIGUIENTE_SESION.md` con un resumen de dónde nos quedamos y los 3 próximos pasos lógicos para arrancar la siguiente sesión.

Confirma al usuario cuando hayas terminado el protocolo para que pueda cerrar la terminal.

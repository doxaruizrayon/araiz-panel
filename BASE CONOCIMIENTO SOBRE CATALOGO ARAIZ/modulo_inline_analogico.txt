¿QUÉ ES Y PARA QUÉ SIRVE?
--------------------------------------------------------------
Este módulo inline de salida analógica es el componente que permite al sistema
I/O Inline de Phoenix Contact enviar señales de corriente 4-20 mA a actuadores
de campo como válvulas proporcionales, variadores de velocidad o posicionadores.
La versión ECO es la variante económica y compacta del sistema Inline, optimizada
para aplicaciones donde se necesita control analógico de corriente sin funciones
avanzadas adicionales. Tiene 4 canales de salida de corriente.

Lo que facilita:
- Controlar de forma proporcional hasta 4 actuadores de campo con señal 4-20mA
  desde el sistema Inline del PLC.
- Integrar el control analógico en una estación Inline junto con módulos digitales,
  sin necesidad de un sistema separado.
- Detectar cable roto en cada canal (si la carga presenta circuito abierto).

Lo que evita que ocurra:
- Que el sistema Inline quede sin capacidad de salida analógica (se añade este
  módulo al conjunto de I/O existente).
- Control todo/nada de actuadores que necesitan regulación proporcional.
- Perturbaciones por ruido eléctrico: la señal 4-20mA es inmune al ruido.

PREGUNTAS QUE TIENES QUE HACER AL CLIENTE
--------------------------------------------------------------
| #  | Pregunta                                                          | Por qué la necesitas                                                          |
|----|-------------------------------------------------------------------|-------------------------------------------------------------------------------|
| 1  | ¿Qué sistema I/O tiene? (Inline IB IL, Axioline AXL...)           | El módulo debe ser de la misma serie que el acoplador instalado               |
| 2  | ¿Cuántas salidas analógicas de corriente necesita?                | 4 canales por módulo — calcular cuántos módulos hacen falta                   |
| 3  | ¿Los actuadores admiten señal 4-20mA?                             | Si admiten 0-10V → necesita el módulo de voltaje, no este                    |
| 4  | ¿A qué distancia están los actuadores del cuadro?                 | 4-20mA es buena para distancias largas — confirmar si hay ruido eléctrico     |
| 5  | ¿Necesita diagnóstico de canal (detección de cable roto)?         | Si sí → versión full en vez de ECO                                           |
| 6  | ¿Es instalación nueva o ampliación de un sistema Inline?          | Si es ampliación → misma serie y versión de protocolo                        |

¿QUÉ ES Y PARA QUÉ SIRVE?
--------------------------------------------------------------
Un módulo de seguridad PSR es el componente que garantiza la seguridad de
personas y máquinas supervisando los dispositivos de seguridad de una instalación:
setas de emergencia, barreras de luz, puertas de seguridad, alfombras sensitivas,
mandos a dos manos... Cuando uno de estos dispositivos se activa (alguien pulsa la
seta de emergencia), el módulo corta de forma segura y certificada la alimentación
de los actuadores peligrosos (motores, prensas, robots), evitando lesiones.

A diferencia de un relé o PLC convencional, este módulo está certificado para
funciones de seguridad hasta SIL 3 / PL e — el nivel más alto de seguridad
funcional reconocido por las normas europeas de maquinaria.

Lo que facilita:
- Supervisar múltiples dispositivos de seguridad en una sola unidad configurable.
- Cumplir la normativa de seguridad de maquinaria (Directiva Maquinaria, EN ISO 13849,
  EN IEC 62061) sin necesidad de diseñar circuitos de seguridad complejos.
- Ampliar fácilmente el sistema añadiendo módulos de extensión vía TBUS sin cableado adicional.
- Configurar las funciones de seguridad por software sin modificar el hardware.

Lo que evita que ocurra:
- Accidentes graves por fallo en la detención de maquinaria peligrosa.
- Que un único fallo en el sistema de control cause una situación peligrosa
  (el módulo tiene redundancia interna y autodiagnóstico).
- Incumplimiento de la normativa CE de maquinaria (multas, retirada del mercado).
- Reinicio accidental de la máquina tras una parada de emergencia sin rearmado manual.

DIFERENCIA CLAVE CON OTROS MÓDULOS DE SEGURIDAD
--------------------------------------------------------------
| Tipo                            | Para qué sirve                                   | Nivel seguridad | Cuándo usarlo                                      |
|---------------------------------|--------------------------------------------------|-----------------|----------------------------------------------------|
| Relé de seguridad simple        | Una sola función (seta emergencia o puerta)      | SIL 2 / PL d    | Máquinas simples con un solo punto de seguridad    |
| Módulo PSR configurable (base)  | Múltiples funciones configurables por software   | SIL 3 / PL e    | Máquinas complejas. El ref. 1104975 es este tipo   |
| Módulo PSR de extensión         | Amplía entradas/salidas del módulo base          | SIL 3 / PL e    | Cuando se necesitan más I/O de seguridad (ref. 1104887) |
| Interfaz de datos PSR           | Conecta el sistema PSR al PLC / red industrial   | —               | Para diagnóstico remoto desde SCADA (ref. 1105102) |
| Safety PLC completo             | PLC que integra lógica estándar y de seguridad   | SIL 3 / PL e    | Líneas muy complejas con muchos ejes y funciones   |

| Configuración                | Cómo se programa                                 | Software PSRwin (sin hardware adicional)      |

PREGUNTAS QUE TIENES QUE HACER AL CLIENTE
--------------------------------------------------------------
| #  | Pregunta                                                          | Por qué la necesitas                                                          |
|----|-------------------------------------------------------------------|-------------------------------------------------------------------------------|
| 1  | ¿Qué dispositivos de seguridad tiene la máquina?                  | Seta emergencia / puerta / barrera / alfombra → define entradas necesarias    |
| 2  | ¿Cuántos dispositivos de seguridad hay en total?                  | Para saber si necesita módulo base solo o también módulos de extensión        |
| 3  | ¿Qué nivel de seguridad exige la máquina? (SIL / PL)              | Lo determina la evaluación de riesgo. Si no lo sabe → derivar al fabricante  |
| 4  | ¿Necesita diagnóstico remoto del sistema de seguridad?            | Si sí → añadir la interfaz de datos PROFINET (ref. 1105102)                  |
| 5  | ¿Qué PLC o red industrial tiene la instalación?                   | Para elegir la interfaz de datos correcta (PROFINET, EtherNet/IP, PROFIBUS)  |
| 6  | ¿El rearme tras emergencia debe ser manual o automático?          | Norma: siempre manual supervisado. Confirmar con el cliente                  |
| 7  | ¿Es instalación nueva o sustitución de un relé de seguridad?      | Si es sustitución → verificar que el nivel de seguridad es equivalente       |

CONSEJO: Los módulos de seguridad siempre requieren evaluación de riesgo previa
(Anexo I Directiva Maquinaria). Si el cliente no tiene esto claro, recomendarle
que contacte con un integrador de seguridad certificado antes de presupuestar.

¿QUÉ ES Y PARA QUÉ SIRVE?
--------------------------------------------------------------
Un módulo de ampliación PSR es el componente que se añade al módulo base de
seguridad (ref. 1104975) cuando la instalación tiene más dispositivos de seguridad
de los que el módulo base puede gestionar por sí solo. Se conecta directamente
sobre el carril DIN junto al módulo base mediante el conector TBUS interno,
sin necesidad de cableado adicional entre módulos. Cada módulo de extensión
añade 16 entradas seguras adicionales al sistema.

Lo que facilita:
- Escalar el sistema de seguridad a medida que crece la instalación, añadiendo
  módulos uno a uno sin rediseñar el sistema.
- Conectar hasta 18 módulos de extensión a un único módulo base, cubriendo
  instalaciones con muchos puntos de seguridad.
- Reducir el cableado entre módulos: la conexión es mecánica vía TBUS en el carril.
- Mantener el mismo nivel de certificación SIL 3 / PL e en todas las entradas ampliadas.

Lo que evita que ocurra:
- Tener que cambiar el módulo base por uno más grande cuando la instalación crece.
- Cableado de señales de seguridad entre módulos (con el TBUS es todo interno).
- Pérdida del nivel de seguridad al ampliar (la extensión mantiene la misma certificación).
- Instalar un segundo sistema de seguridad independiente por falta de entradas.

DIFERENCIA CLAVE CON OTROS MÓDULOS DE EXTENSIÓN PSR
--------------------------------------------------------------
| Tipo                              | Qué añade                          | Conector TBUS incluido | Cuándo usarlo                                  |
|-----------------------------------|------------------------------------|------------------------|------------------------------------------------|
| PSR-M-EF2-SDI16-PI (ref. 1104887) | 16 entradas seguras adicionales    | Incluido               | Ampliar entradas cuando el base se queda corto |
| Módulo extensión salidas seguras  | Salidas seguras adicionales        | Incluido               | Cuando se necesitan más cortes de potencia     |
| Módulo extensión mixto I/O        | Entradas + salidas adicionales     | Incluido               | Instalaciones mixtas con muchos I/O de seguridad|
| Módulo TBUS pasivo                | Solo transmite datos entre módulos | —                      | Separar físicamente módulos en el carril       |

============================================================

TIPOLOGÍAS — VER CUADRO MÓDULO DE SEGURIDAD PSR (ref. 1104975)
Este módulo siempre va acompañado del módulo base PSR. Sin el módulo base,
el módulo de extensión no funciona de forma independiente.

============================================================

PREGUNTAS QUE TIENES QUE HACER AL CLIENTE
--------------------------------------------------------------
| #  | Pregunta                                                          | Por qué la necesitas                                                          |
|----|-------------------------------------------------------------------|-------------------------------------------------------------------------------|
| 1  | ¿Ya tiene instalado el módulo base PSR? (ref. 1104975)            | Sin el módulo base este módulo no funciona — siempre van juntos               |
| 2  | ¿Cuántos dispositivos de seguridad adicionales necesita conectar?  | 16 entradas por módulo de extensión — calcular cuántos hacen falta            |
| 3  | ¿Tiene suficiente espacio en el carril DIN?                        | Cada módulo ocupa espacio en el armario — verificar antes de presupuestar     |
| 4  | ¿El conector TBUS del módulo base está disponible?                 | Si el TBUS está ocupado o no es compatible → verificar con Phoenix Contact    |
| 5  | ¿Es instalación nueva o ampliación de un sistema PSR existente?   | Si es ampliación → verificar versión del firmware del módulo base             |

CONSEJO: Siempre presupuestar el módulo base (1104975) y los módulos de extensión
(1104887) juntos. Un cliente que pide solo el módulo de extensión sin el base
no podrá usarlo — preguntar siempre si ya tiene el sistema PSR instalado.
